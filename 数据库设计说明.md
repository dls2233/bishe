# 校园安全教育平台数据库设计说明

本系统采用 MySQL 8.0 作为关系型数据库，数据库名称为 `campus_security`。系统数据层基于实体关系模型（ER）进行设计，采用 InnoDB 存储引擎以支持事务管理（特别是在积分兑换等核心业务中）。

本系统共设计了 10 张核心业务表，以下是各表的详细结构与字段说明。

---

## 1. 用户表 (`sys_user`)
用于存储平台用户（学生、管理员）的基本信息和总积分。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 用户唯一标识 |
| `username` | VARCHAR(50) | 非空, 唯一 | 登录账号 |
| `password` | VARCHAR(100) | 非空 | 登录密码（MD5加密） |
| `real_name` | VARCHAR(50) | 可空 | 用户真实姓名 |
| `role` | VARCHAR(20) | 默认 'USER' | 角色标识：`USER` (普通用户), `ADMIN` (管理员) |
| `points` | INT | 默认 0 | 用户当前可用总积分 |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 账号创建时间 |
| `update_time` | DATETIME | 自动更新 | 账号信息最后修改时间 |

## 2. 课程表 (`sys_course`)
用于存储安全教育图文与视频课程的基础数据。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 课程唯一标识 |
| `title` | VARCHAR(100) | 非空 | 课程标题 |
| `category` | VARCHAR(50) | 可空 | 课程分类（如：消防安全、交通安全等） |
| `cover_url` | VARCHAR(255) | 可空 | 课程封面图片链接 |
| `video_url` | VARCHAR(255) | 可空 | 视频播放链接（为空则为纯图文课程） |
| `content` | TEXT | 可空 | 课程图文详细内容 |
| `reward_points` | INT | 默认 10 | 学习完该课程可获得的奖励积分 |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 课程发布时间 |

## 3. 学习进度表 (`sys_course_progress`)
用于记录用户对具体课程的学习状态，防止重复增加积分。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 进度记录ID |
| `user_id` | BIGINT | 非空 | 关联 `sys_user` 表的ID |
| `course_id` | BIGINT | 非空 | 关联 `sys_course` 表的ID |
| `status` | VARCHAR(20) | 默认 'LEARNING' | 状态：`LEARNING` (学习中), `COMPLETED` (已完成) |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 开始学习时间 |
| `update_time` | DATETIME | 自动更新 | 完成学习时间 |
*联合唯一索引：`uk_user_course (user_id, course_id)`，确保一个用户对一门课只有一条进度记录。*

## 4. 试卷表 (`sys_exam`)
用于存储在线测评的试卷基础信息。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 试卷唯一标识 |
| `title` | VARCHAR(100) | 非空 | 试卷名称 |
| `description` | VARCHAR(255) | 可空 | 试卷的说明和考试须知 |
| `total_score` | INT | 默认 100 | 试卷满分 |
| `pass_score` | INT | 默认 60 | 及格分数线 |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 试卷创建时间 |

## 5. 试题表 (`sys_question`)
用于存储具体试卷下的题目明细。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 试题唯一标识 |
| `exam_id` | BIGINT | 非空 | 归属的试卷ID，关联 `sys_exam` |
| `content` | TEXT | 非空 | 题目具体内容 |
| `type` | VARCHAR(20) | 非空 | 题型：`SINGLE_CHOICE`, `MULTIPLE_CHOICE`, `JUDGE` |
| `options` | JSON | 可空 | 选择题的选项列表，存储为 JSON 数组格式 |
| `answer` | VARCHAR(255) | 非空 | 正确答案（多选以逗号分隔，如 "A,B"） |
| `score` | INT | 默认 5 | 本题分值 |

## 6. 考试记录表 (`sys_exam_record`)
记录用户参与在线测评的得分及是否及格。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 记录ID |
| `user_id` | BIGINT | 非空 | 考试用户ID |
| `exam_id` | BIGINT | 非空 | 参加的试卷ID |
| `score` | INT | 非空 | 用户最终得分 |
| `is_pass` | TINYINT(1) | 非空 | 是否及格（1: 及格, 0: 不及格） |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 交卷时间 |

## 7. 积分商城商品表 (`sys_goods`)
用于展示供用户兑换的奖品列表。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 商品ID |
| `name` | VARCHAR(100) | 非空 | 商品名称 |
| `description` | VARCHAR(255) | 可空 | 商品详情描述 |
| `image_url` | VARCHAR(255) | 可空 | 商品图片链接 |
| `points_required` | INT | 非空 | 兑换该商品所需的积分额度 |
| `stock` | INT | 默认 0 | 当前剩余库存 |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 商品上架时间 |

## 8. 积分兑换记录表 (`sys_exchange_record`)
记录用户的商品兑换流水，作为发货凭证。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 兑换流水ID |
| `user_id` | BIGINT | 非空 | 兑换用户ID |
| `goods_id` | BIGINT | 非空 | 兑换的商品ID |
| `points_cost` | INT | 非空 | 实际消耗的积分数 |
| `status` | VARCHAR(20) | 默认 'PENDING' | 订单状态：`PENDING`(待发货), `COMPLETED`(已完成) |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 兑换操作时间 |

## 9. 新闻资讯表 (`sys_news`)
用于展示校园内最新的安全动态。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 资讯ID |
| `title` | VARCHAR(100) | 非空 | 资讯标题 |
| `category` | VARCHAR(50) | 可空 | 资讯分类 |
| `content` | TEXT | 非空 | 新闻图文正文 |
| `cover_url` | VARCHAR(255) | 可空 | 新闻封面 |
| `views` | INT | 默认 0 | 阅读量统计 |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 发布时间 |

## 10. 安全预警表 (`sys_alert`)
用于支撑 SSE 实时预警推送的持久化数据表。
| 字段名 | 数据类型 | 约束 | 描述 |
| --- | --- | --- | --- |
| `id` | BIGINT | 主键, 自增 | 预警记录ID |
| `title` | VARCHAR(100) | 非空 | 预警弹窗标题 |
| `content` | TEXT | 非空 | 预警详情内容 |
| `level` | VARCHAR(20) | 默认 'INFO' | 危险等级：`INFO`(通知), `WARNING`(警告), `DANGER`(危险) |
| `status` | VARCHAR(20) | 默认 'ACTIVE' | 当前状态：`ACTIVE`(生效中), `RESOLVED`(已解除) |
| `create_time` | DATETIME | 默认 CURRENT_TIMESTAMP | 预警发布时间 |
